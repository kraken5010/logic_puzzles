The application "Logic Puzzles" was created to develop the creative thinking of users.

Written in Django using PostgreSQL database. Neumorphism was used on the frontend.

Implemented functionality:

  • Customization of the User model has been completed using AbstractUser.
    Added new user fields;
  • User registration;
  • User authorization;
  • Automatic authorization after successful registration;
  • Adding custom puzzles;
  • Field validation for abusive words;
  • Inability to add new puzzles without authorization;
  • Getting previous and next tasks from the database;
  • Adding comments by users;
  • Commenting on comments from previous users;
